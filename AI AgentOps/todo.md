- **프론트엔드:** React 19, Tailwind 4, shadcn/ui, Recharts

- **데이터베이스:** MySQL/TiDB
- **인증:** Manus OAuth
- **AI:** Google Gemini 2.0 API
- **모니터링:** Sentry, PostHog
- **배포:** Vercel
- **테스트:** Vitest
